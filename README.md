## 紹介と使い方

- 「チーズアカデミー」LPを忠実再現＆JOJO絵描きアカデミーという架空スクールのLPサイト

- 一式ローカルに配置し、index.html index2.html をブラウザで開けば閲覧可能

## 工夫した点

- ホバーアクションでエリア全体の背景を反転
- 画像スライドショーで２枚ずつ表示
- 会社情報のフォームをカスタマイズ
- できたhtmlをコピペして自分の好きな情報に置き換え

## 苦戦した点

- 画像が交互に並んでいる部分のテキスト上下余白が意外と苦戦
- 画像スライドがうまく表示されず苦戦
